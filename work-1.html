New Balance DMS

Redesign the website and add other features

About this project

Overview
It is used to order related products that are needed for in-store furnishings, such as hangers, insoles, display racks... etc.
The client wants to redesign the webpage and add other features.
Problem
- The webpage does not have a responsive design.
- The information structure of the old website is too messy, and users are easily lost in the website.
Solution
Focus on easy-to-read website information, re-adjust the information structure of the web page.
Users
The users are internal employees in Asia.
